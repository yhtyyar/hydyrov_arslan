# Hydyrov Arslan — Стоматологический лендинг

Статический лендинг для врача-стоматолога **Хыдырова Арслана**. Развёрнут на GitHub Pages.

**🌐 Сайт:** https://yhtyyar.github.io/hydyrov_arslan/

---

## 🏗 Технологии

| Компонент | Технология |
|-----------|------------|
| **Framework** | Next.js 14 (Static Export) |
| **Styling** | TailwindCSS |
| **UI** | shadcn/ui |
| **Deploy** | GitHub Pages |
| **CI/CD** | GitHub Actions |

---

## 📁 Структура проекта

```
hydyrov_arslan/
├── web/                    # Next.js приложение
│   ├── src/
│   │   ├── app/            # Страницы (лендинг, портфолио, отзывы)
│   │   └── components/     # React-компоненты
│   └── public/             # Статика (изображения в ./images/)
├── .github/workflows/       # GitHub Actions для деплоя
├── photos/                  # Исходные фото (не в коммитах)
└── README.md
```

---

## 🚀 Быстрый старт

### Локальная разработка

```bash
cd web
npm install
npm run dev
```

### Сборка для продакшена

```bash
cd web
npm run build
```

Статические файлы собираются в `web/dist/`.

---

## 📸 Изображения

### Формат
- WebP для оптимизации размера
- Размещаются в `web/public/images/`
- Пути в коде: `./images/имя.webp` (относительные)

### Источник
Исходные фото хранятся в `photos/webp/` и копируются в `web/public/images/` перед деплоем.

---

## 🔗 Поведение при клике

### Главная страница
- **Карточки портфолио** → переход на страницу `/portfolio`

### Страница портфолио
- **Карточка работы** → открывается модальное окно с каруселью всех фото работы
- **Стрелки** → переключение между фото
- **Миниатюры** → быстрый переход к конкретному фото
- **Крестик или фон** → закрыть модальное окно
