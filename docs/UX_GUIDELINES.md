# UX/UI Рекомендации для стоматологической клиники

## 1. Цветовая палитра

### Основная палитра (Trust & Clean)

| Роль | Цвет | HEX | Обоснование |
|---|---|---|---|
| **Primary** | Тёмно-бирюзовый | `#0D9488` | Ассоциация с медициной, чистотой, спокойствием |
| **Primary Light** | Светлая бирюза | `#5EEAD4` | Акценты, hover-состояния |
| **Secondary** | Тёплый золотистый | `#D97706` | Премиальность, доверие |
| **Background** | Чистый белый | `#FFFFFF` | Стерильность, простор |
| **Surface** | Светло-серый | `#F8FAFC` | Карточки, секции |
| **Text Primary** | Тёмно-графитовый | `#1E293B` | Читаемость |
| **Text Secondary** | Серый | `#64748B` | Подписи, мета-текст |
| **Success** | Зелёный | `#16A34A` | Подтверждение записи |
| **Error** | Красный | `#DC2626` | Ошибки, отмена |

### Запрещённые цвета
- ❌ Яркий красный как основной (ассоциация с кровью/болью)
- ❌ Тёмные/мрачные фоны (ассоциация с тревожностью)
- ❌ Кислотные/неоновые оттенки (несерьёзность)

## 2. Типографика

- **Заголовки:** Inter или Manrope (современные, чистые)
- **Основной текст:** Inter, 16px, line-height: 1.6
- **Минимальный размер:** 14px (доступность)

## 3. Паттерны доверия в стоматологии

### Hero-секция
- Профессиональное фото врача (в форме, с улыбкой)
- Краткая подпись: имя, специализация, опыт
- CTA: «Записаться на приём» (крупная, контрастная кнопка)

### Социальное доказательство
- Счётчик: «500+ довольных пациентов»
- Рейтинг: ★★★★★ (средний балл)
- Отзывы с фото пациентов (с разрешения)
- Логотипы сертификатов / учебных заведений

### Портфолио «До/После»
- Слайдер с разделителем (перетаскиваемый)
- Фильтры по категориям: виниры, имплантация, отбеливание
- Подписи: диагноз → решение → результат

### Блок записи
- Минимум полей (имя, телефон, желаемая дата)
- Визуальный календарь со свободными слотами
- Зелёные слоты = свободно, серые = занято
- Подтверждение: «Вы записаны на [дата] в [время]»

## 4. Паттерны UX

### Мобильный приоритет (Mobile-First)
- 70%+ трафика — со смартфонов
- Кнопка «Позвонить» и «WhatsApp» — фиксированные внизу
- Упрощённая навигация: бургер-меню, максимум 5 пунктов

### Конверсионные элементы
- Sticky CTA-кнопка «Записаться» при скролле
- Popup после 30 сек на сайте: «Запишитесь сегодня — скидка 10%»
- Микроанимации при hover (плавные, не агрессивные)

### Формы
- Максимум 3-4 поля для первичной записи
- Маска ввода для телефона (+7 XXX XXX XX XX)
- Inline-валидация (мгновенная, без перезагрузки)
- Кнопка отправки — всегда видна (не уезжает за экран)

## 5. Иконография

- Стиль: линейные (outline), stroke: 1.5-2px
- Библиотека: **Lucide Icons** (согласованность с shadcn/ui)
- Медицинские иконки: зуб, улыбка, щит (безопасность), календарь

## 6. Анимации

- **Длительность:** 200-300ms (быстрые, не раздражающие)
- **Easing:** `ease-out` для появления, `ease-in` для скрытия
- **Где применять:**
  - Появление секций при скролле (Intersection Observer)
  - Hover на кнопках и карточках
  - Переход между шагами записи
- **Где НЕ применять:**
  - Критические действия (подтверждение записи)
  - Загрузка медицинских данных

## 7. Доступность (a11y)

- Контрастность текста: минимум 4.5:1 (WCAG AA)
- Все интерактивные элементы — с фокусом (outline)
- Alt-текст для всех изображений
- Поддержка навигации с клавиатуры
